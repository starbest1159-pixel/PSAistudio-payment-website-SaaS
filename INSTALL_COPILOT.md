# คู่มือการติดตั้ง GitHub Copilot CLI

> **ความเป็นเจ้าของ:** เอกสารนี้และสิทธิ์ในการบริหารจัดการกระบวนการติดตั้งนี้เป็นของ **@255yxtaf** เท่านั้น

## บทนำ

เอกสารนี้จัดทำขึ้นเพื่อให้คำแนะนำเกี่ยวกับการติดตั้ง GitHub Copilot Command Line Interface (CLI) สำหรับโปรเจกต์นี้

## ขั้นตอนการติดตั้ง

รันคำสั่งต่อไปนี้ในเทอร์มินัลของคุณ:

```bash
curl -fsSL https://gh.io/copilot-install | bash
```

## การตรวจสอบการติดตั้ง

หลังจากติดตั้งเสร็จสิ้น ให้รันคำสั่งเพื่อตรวจสอบเวอร์ชัน:

```bash
export PATH=$PATH:/home/ubuntu/.local/bin
copilot --version
```
